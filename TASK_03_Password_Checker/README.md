# Task 03: Password Strength Checker

## Description
This Python script checks the strength of a given password based on criteria like length, presence of uppercase and lowercase letters, numbers, and special characters. It provides feedback to the user on the password's strength.

## Files
- **password_checker.py**: The main script for checking password strength.

## How to Run
1. Run the script using Python:
    ```bash
    python password_checker.py
    ```
2. Enter the password to check when prompted.

## Input
- **Password**: The password to evaluate.

## Output
- The strength of the password: **Weak**, **Moderate**, or **Strong**.
- Feedback on which criteria were not met.

## Requirements
- Python 3.x

## Notes
- This script is intended for educational purposes and basic security checks.
